# InsightFlow

InsightFlow 是一个 Codex Skill，用于将行业热点在 5 分钟内转化为结构化研报。输入关键词或关键词+链接，输出对齐背景、权威信源分级、观点对比，并推荐 3 条核心阅读。

## 适用范围

AI、ESG 合规、金融投资、半导体、互联网等行业热点解读。

## 使用方式

将 `insight-flow/` 放入你的 Codex skills 目录，按需触发即可。典型触发方式：
- 仅提供关键词
- 关键词 + 1-5 个链接

## 输出结构

1. 执行摘要
2. 重要性与影响
3. 背景快照（技术/历史/监管）
4. 信源分级
5. 核心论点与证据
6. 观点分歧
7. 未解问题与观察清单
8. 核心阅读推荐（正好 3 条链接）
